== Description ==

Add more Social Icons to Mailpoet Plugin, For activating this plugin you first you need to install Mailpoet Newsletter and activate it, this plugin won't activate until mailpoet completely installed and activated.
after that you good to go. just use Mailpoet the normal way but in Social bookmarks of newsletter you have a massive medium sized icons. if you have another icon in your mind or additional social icons to be added in plugin don't hesitate and send them to me at: info@sepideman.com, info@securitexperts.com, info@zartoshtsepideman.com.

== Installation ==

1. First download and install `wysija-newsletters` plugin
1. Activate `Mailpoet newsletter` Plugin
1. Second download and install `sepideman-mailpoet-social-icons` plugin
1. At last activate Mailpoet Sepideman Social Icons` Plugin

1. ابتدا پوشه `wysija-newsletters` را در پوشه افزونه‌های وبسایت خود بریزید.
1. افزونه `Mailpoet Newsletter` را فعال کنید.
1. سپس پوشه `sepideman-mailpoet-social-icons` را در پوشه افزونه های خود بریزید.
1. سپس افزونه `Mailpoet Sepideman Social Icons` را فعال کنید.
